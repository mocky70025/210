# ポイント制ミニゲームプラットフォーム

身内限定のポイント制ミニゲームプラットフォームです。

## 技術スタック

### フロントエンド
- React + TypeScript
- Vite
- Tailwind CSS

### バックエンド
- Node.js + TypeScript
- Express
- Socket.IO

## セットアップ

### 1. バックエンドのセットアップ

```bash
cd server
npm install
npm run dev
```

サーバーは `http://localhost:3000` で起動します。

### 2. フロントエンドのセットアップ

別のターミナルで：

```bash
cd client
npm install
npm run dev
```

フロントエンドは `http://localhost:5173` で起動します。

## 使い方

1. ブラウザで `http://localhost:5173` にアクセス
2. ログイン画面で以下を入力：
   - ユーザー名：自由に入力（他のユーザーと重複しないように）
   - パスワード：`only-friends-2025`（共通パスワード）
3. ログイン後、ロビー画面で以下が可能：
   - オンラインユーザー一覧の確認
   - ポイント譲渡（他のユーザーにポイントを送る）
   - ルーム作成・参加
4. じゃんけんゲーム：
   - ルーム作成または参加
   - 2人揃ったら自動的にゲーム開始
   - 手を選択（グー・チョキ・パー）
   - 勝敗に応じてポイントが増減
   - 引き分けの場合は再戦

## 機能

- ✅ ログイン（共通パスワード方式）
- ✅ ポイント管理（初期1000pt）
- ✅ オンライン対戦（最大8人同時接続）
- ✅ じゃんけんゲーム（100pt固定ベット）
- ✅ ポイント譲渡機能
- ✅ リアルタイム通信（Socket.IO）

## ゲームルール

### じゃんけん
- ベット額：100pt（固定）
- 2人対戦
- 引き分けの場合：ポイント変動なし、再戦
- 勝者：100pt獲得
- 敗者：100pt減少

## 注意事項

- ポイントはゲーム内通貨のみで、現金・電子マネーなどには交換できません
- サーバー再起動時は、すべてのデータ（ポイント・ルーム情報）がリセットされます
- 同じユーザー名で複数回ログインすることはできません
