### 2. プロジェクトのルートディレクトリに移動（オプション）

## 実行結果

成功すると、以下のようなJSONが返されます：

```json
{
  "success": true,
  "message": "ユーザー登録が完了しました",
  "userId": 1
}
```

エラーの場合は、エラーメッセージが表示されます。

## 注意事項

- **管理パスワード**（`adminPassword`）は、Renderの環境変数 `ADMIN_PASSWORD` と一致している必要があります
- デフォルトは `admin-secret-2025` です
- 環境変数を変更した場合は、その値を使用してください

## トラブルシューティング

### 「ユーザー登録は運営のみが行えます」と表示される場合

- `adminPassword` が正しく設定されているか確認
- Renderの環境変数 `ADMIN_PASSWORD` を確認

### 接続エラーが発生する場合

- Renderのバックエンドが起動しているか確認
- URLが正しいか確認（`https://two10-u8xp.onrender.com`）
